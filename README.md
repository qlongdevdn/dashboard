# Dashboard Systems' DevDaNang Services

This is the backend systems of Dashboard Quang Long Systems.

## Introduction

Using NodeJs and MongoDB ( Mongoose )

## Install and follow step

```bash

# Genarate key JWT in systems

cd keystore
./generator.sh
cd ../

# Install package

npm install
```

## Create .env file

```
APP_PORT=""

APP_URL=""

MONGO_HOST=""

MONGO_DB=""
MONGO_USER=""
```
